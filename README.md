# Loja de Jogos (Projeto Integrador)
Projeto educacional do Senac EAD voltado para o desenvolvimento de um sistema de uma loja de jogos.

# Status do Projeto
Em desenvolvimento.

# Tecnologias que serão aplicadas
- Java
- Banco de Dados (Mysql)
- JPA (Java Persistence API)

# Time de desenvolvedores
- Nicholas Dela Pace Cabrera (Laura Dela Pace Cabrera)

# Objetivo do software
O sistema tem como objetivo principal de gerenciar a locação e venda de jogos de uma loja e armazenar estas informações.

# Funcionalidades do Sistema
- Cadastrar jogos (para venda ou locação)
- Cadastrar clientes (vinculados às transações)
- Criar categorias (vinculadas aos jogos)
- Cadastrar funcionários (usuários do sistema)
- Realizar login com validação de usuário e senha
- Registrar locações
- Registrar compras
- Consultar jogos (produtos)
- Gerenciar carrinho (adicionar/remover jogos de compra ou locação)
- Calcular valor total da transação
- Excluir jogos registrados
- Definir data de devolução
- Calcular atraso
- Buscar jogos por nome
- Buscar jogos por tipo (locável/comprável)
